# 🎨 Gerador de Paleta de Cores

Um gerador de paletas de cores aleatórias feito com HTML, CSS e JavaScript puro.

## > Como usar

1. Abra o arquivo `index.html` no navegador
2. Clique em **"Gerar Paleta"** para criar 5 cores aleatórias
3. Clique em qualquer cor para copiar o código hex para a área de transferência

## > Funcionalidades

- Gera 5 cores aleatórias com um clique
- Exibe o código hex de cada cor
- Copia o código da cor ao clicar no card
- Animações suaves ao passar o mouse

## > Tecnologias utilizadas

- HTML
- CSS3
- JavaScript
